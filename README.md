# Tissue Agent Simulator

A simulation framework for AI-driven pharmaceutical research, exploring how autonomous agents can model and optimize preclinical testing with lab-grown human tissues. This project investigates the intersection of computational agents, biological systems, and drug development workflows.

## Key Features
*   Multi-agent system architecture for simulating experimental design and analysis.
*   Integration of biological tissue response models within an agent decision loop.
*   Configurable simulation environments for in silico testing of research protocols.
*   Data logging and visualization for agent behavior and simulated outcomes.

## Tech Stack
*   Python
*   LangChain
*   FastAPI
*   Docker

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/tissue-agent-simulator.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the simulation: `python main.py`
